## Link

`AlertLink` is to provide matching colored links within `Alert`.

Use `render` prop to render component other than HTML `a`, such as `Link` of React Router.

<Playground>
  <Alert theme="primary">
    A simple primary alert with <AlertLink href="javascript:">an example link</AlertLink>. Give it a click if you like.
  </Alert>
  <Alert theme="primary">
    A simple primary alert with <AlertLink render={props => <Link {...props} to="/">a router link</Link>} />. Give it a click if you like.
  </Alert>
</Playground>
